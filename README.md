# MoodLine Pro 

我打造的對話分析工具，解開對話背後的隱藏真相。

---

## 核心特色

- 深度數據分析：話量比例、回覆速度、深夜活躍度
- 通話行為通靈：統計通話長度、頻率
- 互動文字雲：一眼看出對話中的高頻關鍵字
- 100% 隱私保護：數據純本地運算，絕不上傳雲端

---

## 快速開始

### 1. 匯出聊天紀錄 (.txt)
*   手機版 LINE：進入對話 -> 右上選單 -> 其他設定 -> 傳送聊天紀錄。

### 2. 上傳並分析
*   將匯出的 .txt 檔案直接拖入網頁，即可瞬間生成全方位報告。
